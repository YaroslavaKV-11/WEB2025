Для перевірки роботи функції відкрийте консоль та запускайте функцію isString()

Приклади виконання функції

// Рядок (примітив)
isString('hello')               // Очікується: true

// Рядок (об'єкт)
isString(new String('hello'))   // Очікується: true

// Не рядки
isString(123)                   // Очікується: false
isString(true)                  // Очікується: false
isString(null)                  // Очікується: false
isString(undefined)             // Очікується: false
isString(['a', 'b'])            // Очікується: false
isString({text: 'hi'})          // Очікується: false
isString(Symbol('s'))           // Очікується: false
